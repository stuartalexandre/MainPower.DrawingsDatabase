Installing

Using the installer

Run the DrawingDatabaseSetup.msi installer file, and follow the prompts. An icon will be added to your desktop to the main program, put this where you wish.

Using the zip file

Extract the contents of the zip file to a folder of your choice. Run DrawingsDatabaseGUI.exe to show the main search window.

From source

Extract the DrawingsDatabaseSource.zip file to a folder of your choice. Open the solution file with visual studio (or C# Express Edition) and hit build.
