Using the AutoCAD Interface

Note: Before you can use the AutoCAD interface, you must first install the AutoCAD plugin. You can do this by selecting Tasks->AutoCAD Plugin from the Drawings Database program.

You can update drawing title blocks from the database and vice-versa using the AutoCAD interface to the database, which is implemented via the AutoCadPlugin.dll file. The plugin exposes the following functions to AutoCAD:
  • ddbget - Pull data from the database into a selected block
  • ddbgetd - Pull data from the database into all blocks in the drawing
  • ddbgetl - Pull data from the database into all blocks in the current layout
  • ddbput - Put data from the selected block into the database
  • ddbputd - Put data from all the blocks in the drawing into the database
  • ddbputl - Put data from all the blocks in the current layout into the database
  • ddbsettings - Edit various settings used by the library
  • ddbsearch - Show the search and editing dialog for the database
  • ddbabout - Shows the about information dialog
  • ddbadd - Manually add a drawing into the database
  • ddbmorerev - Shift the revisions down so that b revision occupies a revision and so on (a drops off the bottom and is lost)
  • ddbnum - Autonumber a title block and save
  • ddbedit - Opens a dialog allowing editing of drawing attributes, the saves to the database and updates the title block.
  • ddbversion - Prints the database version
  • ddblink - Creates an entry in the database for open drawing, fills out the filename automatically but nothing else.  (Designed for drawings with non standard title blocks)
  • ddbrevab - Add revision lebelled AS BUILT.  Will ask for drawn/approved initials.
  • ddbrevifc - Add revision labelled ISSUED FOR CONSTRUCTION.  Will ask for drawn/approved initials.
